# AAS Project for Diss

**Felix Oke** · Information Sciences PhD Student · University of Illinois Urbana-Champaign

Planning and background materials for a developing dissertation collaboration with the **American Antiquarian Society** (AAS, Worcester, MA) on early African American printing and publishing. Advised by **Prof. Ryan Cordell** (School of Information Sciences & English, UIUC, PI of the [Viral Texts](https://viraltexts.org/) project); AAS-side contact is **John J. Garcia**, Director of Scholarly Programs.

This is a planning repo, not a code repo — it tracks the collaboration's scope, relevant AAS holdings, and how it connects to prior work, ahead of an in-progress dissertation.
